# 📊 부동산 매물 데이터 자동 정제 및 엑셀 리포트 생성 도구
# 1. 프로젝트 개요
공공데이터 또는 CSV 형태로 제공되는 부동산 매물 데이터를 자동으로 정제하고,
지역별 및 거래유형별 통계를 계산하여 엑셀 리포트를 생성하는 업무 자동화 프로젝트입니다.

반복적으로 수행되는 데이터 정리 및 보고서 작성 과정을 자동화하여
업무 효율을 향상시키고 데이터 처리 과정에서 발생하는 오류를 줄이는 것을 목표로 합니다.

# 2. 문제 상황
기존에는 매물 데이터를 수동으로 정리하고 필터링하여
엑셀로 보고서를 작성해야 했습니다.

이 과정에서:

데이터 형식이 일정하지 않아 추가 정리가 필요했고
중복 데이터 및 결측 데이터로 인해 오류가 발생했으며
반복적인 작업으로 시간이 많이 소요되었습니다

# 3. 해결 방법
Python과 pandas를 활용하여 데이터 처리 전 과정을 자동화했습니다.

CSV 데이터 자동 로드
중복 및 결측치 제거
데이터 형식 정제 (가격, 지역 등)
지역별 / 거래유형별 통계 자동 계산
엑셀 리포트 자동 생성

# 4. 주요 기능
📥 데이터 자동 불러오기 (CSV / 공공데이터)
🧹 데이터 정제 (중복 제거, 결측치 처리, 형식 통일)
📊 통계 생성
지역별 매물 수
거래유형별 매물 수
평균 가격 계산
📄 엑셀 리포트 자동 생성 (다중 시트)
